About This Test Suite

This documentation contains performance analysis charts comparing system behavior with and without NetFunnel integration. The charts display various metrics including CPU usage, replica counts, request patterns, latency measurements, and network traffic to help analyze the impact of NetFunnel on system performance during autopilot testing scenarios.
